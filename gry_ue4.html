TRIANGLE JUMP

The fourth game I created is based on Doodle Jump.
I added new features such as SaveGame, Particle System (created by me), and Animations.
I made a difficulty system that makes it harder to get easier things as you earn more points.
I also made my own Triangle physics that can be easily changed.
This game took a long time to make, even though it doesn't look like it, because there are many things that are not visible here, such as what happens if the platform spawns too high? What if a broken platform spawns multiple times, etc.
These things took the longest to implement.
The game was created in 20 days (from 31.12.2021 to 20.01.2022).

SOLAR SYSTEM

This is already the third game I have created.
My Physics teacher announced that we had to make a presentation or some other artistic work,
so I thought I would create a game, and that's what it is.
I gathered information about the planets from various sources (mostly Wikipedia),
and it is not copied but changed to look better and be in "one sentence," and it took me the whole day.
I downloaded the planet textures from a website that collaborates with NASA and received these textures from them.
The music playing in the background changes randomly every time you start the game (there are six different tracks).
It was created in about a month.

AUDIR8
